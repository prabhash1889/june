Because voice removes the safety net

Powerful, but never reckless.

You can't "read before you click" when you're talking. So June asks first where it counts.

◆
Spoken confirmation"That will close six terminals. Say confirm to proceed." Destructive actions always ask.
◆
Approvals the brain can't skipConfirmations are one-time tokens verified by the bridge before execution - swapping the model never bypasses a gate.
◆
Asked once, runs onceEvery command carries a unique request id. Retries and reconnects replay the result - they never launch a second batch.
◆
Localhost onlyThe control channel is bound to your machine, token-authed, discovery-verified, and off until you switch it on.
◆
Honest privacy modesStandard, local voice (speech stays on-device), and strict offline (every network capability blocked) - no false "fully offline" claims.
◆
Contained writesFile access stays inside the workspace; parallel write agents get their own Git worktrees or don't launch.
Ten phases, each one shippable

Built in order, so voice is the last easy step.

The plan proves the control loop in text before adding a microphone. Once it works typed, speaking it is mechanical.

Phase 0

Foundations

Tauri shell, tray presence, settings store, keychain, CI. An empty June that launches and remembers.

Phase 1

Bridge control endpoint

A localhost, token-authed channel with a frozen contract: idempotent commands, stable IDs, ordered resumable events.

Phase 2

Control as MCP

Wrap that endpoint as an MCP server - spawn agents, terminals, browser, tasks - callable as tools.

Phase 3

Agent core, typed key milestone

The pluggable brain with capabilities and bridge-verified approvals attached. Type a command, watch it happen exactly once.

Phase 4

Speech to text

June hears you - push-to-talk, VAD, live transcription feeding the agent.

Phase 5

Text to speech

June talks back, streaming as it thinks, and you can interrupt it mid-sentence.

Phase 6

The widget your details

Begins by asking you for the widget spec, then builds the high-quality floating surface to match.

Phase 7

Model choice, settings & privacy modes

Every provider picker, every knob, honest privacy tiers, and the diagnostics that prove it all works.

Phase 8

Wake word & hands-free

"Hey June" - custom wake word, refined turn-taking, reliable low-false-trigger activation.

Phase 9

Expansion & hardening

A non-saple capability to prove it's general-purpose, then recovery drills - killed apps, retries, expired approvals - packaging and polish.
